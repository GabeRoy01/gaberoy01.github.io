Faculty Overview

Faculty machine on HackTheBox, submitted by jbyolo. The machine start from a web server running nginx, eventually leading to a login panel that can be bypassed with SQL injection, then we find a PDF generator that we can inject html into. Wich lead us to LFI, where we can find ssh credentials, after that we can do user pivoting by abusing meta-git to read another user's ssh directory, and finally abuse cap_sys_ptrace to spawn a suid shell.
